# Calculadoras Trabalhistas-py
API desenvolvida em Python para realização de cálculos trabalhistas conforme a legislação brasileira vigente 
(tabelas atualizadas para 2026).  

Esse projeto tem como base a Consolidação das Leis do Trabalho (CLT) e normas complementares.  

Informativo: Projeto ainda em desenvolvimento.

## Objetivo:
Fornecer um conjunto modular de calculadoras trabalhistas reutilizáveis, que possam ser integradas a:  
- Sistemas jurídicos;  
- ERPs;  
- Aplicações web;  
- APIs REST;  
- Ferramentas internas de escritórios.

## Funções já desenvolvidas:
Calculadora de horas extras;  
Calculadora de férias;  
Calculadora de horas;  
Calculadora de décimo terceiro;  
Calculadoras de INSS e IRPF (útiliza como função à parte para demais códigos).

## Funções em desenvolvimento:
Calculadora de recisão contratual.

## Funções a desenvolver:
Calculadora de FGTS;  
Calculadoras de indenizações trabalhistas.  
Calculadora de sálario bruto e líquido;  
Calculadora de correção monetária.
